Dev Tools

Browsers allow you to inspect the code on websites. This is helpfull if you want to check what the styling is on an element, or which element is used. You can access these developer tools by right clicking on a webpage and click "inspect element". You can also press "F12" or on mac "fn+F12".

  1. Right click the "inspect me" button and click inspect form the menu that pops up.
  2. Which class does this button have, which element is used?
  3. Do you see the styling applied to the button
  4. Do you see the hover styling?

Inspect me!

*Note, we use anchors <a> if we want to navigate to another page or to another section within the page, we use <button> when we need quick interactions like play/pause or a hamburger menu.

Transitions tutorial

  1. Style this link with the class `button` so it looks like a button, add padding, a border and a background to it.
  2. Style a hover, where the background and the border color switch.
  3. Add a transition on the button.

This is a button

Hint .button {
    display: inline-flex;
    padding: 10px 20px;
    background: blueviolet;
    color: pink;
    border: 2px solid pink;
    text-decoration: none;
    transition: all 0.3s;
}

.button:hover {
    background: pink;
    border-color: blueviolet;
    color: blueviolet;
}

Animations tutorial

Lets make balls bounce

  1. First step, make it go up and down.
  2. Add some smoothness with easing
  3. Add two more balls
  4. Play around with delay options on the other two balls
Hint @keyframes bounce {
    0% {
        transform: translateY(0);
        opacity: 0.5;
    }
    50% {
        transform: translateY(-50%);
        scale: 1.1;
        opacity: 1;
    }
    100% {
        transform: translateY(0);
        opacity: 0.5;
    }
}

.bouncing-ball {
    width: 100px;
    height: 100px;
    border-radius: 50%;
    background: blueviolet;
    animation: bounce 1s infinite ease-in-out;
}

.bouncing-ball + .bouncing-ball {
    animation-delay: 0.25s;
}

.bouncing-ball + .bouncing-ball + .bouncing-ball {
    animation-delay: 0.5s;
}
