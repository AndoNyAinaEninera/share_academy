Adding event listeners

Documentation

Change content on a click

This is a box that will be a little bit different.
const click = document.getElementById("click");

click.addEventListener("click", function(){
    this.textContent = "clicked it!";
});

You can use your previous variables again.

const click = document.getElementById("click");

click.addEventListener("click", function(){
    this.textContent = title;
});

Add a class on a click.

This is a box that will be a little bit different.
const clickAddClass = document.getElementById("click-add-class");

clickAddClass.addEventListener("click", function(){
    this.classList.add("box-inverse")
});

You can also remove it again.

clickAddClass.addEventListener("click", function(){
    this.classList.remove("box-inverse")
});

Let's toggle this.

clickAddClass.addEventListener("click", function(){
    this.classList.toggle("box-inverse")
});

Create a hamburger menu

We have a navigation and a button to toggle the display of the navigation.

  • Menu item 1
  • Menu item 2
  • Menu item 3

First we need to add some css.

CSS

.main-nav { display: none;}
.main-nav.is-open {display: block}

.hamburger-toggle {
    padding: 0.5em;
    background: var(--js);
    border: 2px solid black;
}

.hamburger-toggle.is-open {
    background: black;
    color: var(--js);
}

Now lets add the JavaScript

JS

const toggleNav = document.getElementById("toggleNav");
const mainNav = document.getElementById("mainNav");

toggleNav.addEventListener("click", function(){
    this.classList.toggle("is-open")
    mainNav.classList.toggle("is-open")
});
