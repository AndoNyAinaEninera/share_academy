Determine the number of bins:
This can be a subjective choice, but some common methods include:
Square root choice: Take the square root of the number of data points and round up. 
Sturges' formula: Number of bins = 1 + log₂(n), where n is the number of data points. 
Freedman-Diaconis' rule: Number of bins = (2 * IQR) / 3√n, where IQR is the interquartile range.
