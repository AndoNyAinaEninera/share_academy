- Math and Date functions

Math operators + - / * % (division between 2 integers will give rounded integer, you need to perform a CAST as REAL on one operand to get REAL value as result, or multiply numerator by 1.0 REAL to transform it to a REAL number)

Math functions
ABS()
CAST()convert an expression to another data type
CAST() can also be used to convert TEXT values into a numerical value like REAL
When doing so, it will only consider any valid prefix at the beginning of the string
for example : SELECT CAST('3.14 is pi' AS REAL); This query will convert the TEXT value '3.14 is pi' into type REAL, considering only the prefix '3.14' and ignoring ' is pi', resulting in 3.14.
